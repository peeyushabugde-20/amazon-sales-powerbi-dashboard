# Amazon Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard that analyzes Amazon sales performance across different product categories, payment methods, and locations.

The dashboard helps identify sales trends, customer payment preferences, and product category contributions.

## Key Features
• Current month orders, quantity, and sales KPIs  
• Product category sales analysis  
• Geographic distribution of sales  
• Sales by payment method  
• Monthly revenue trend analysis  

## Tools Used
• Power BI
• Power Query
• Data Modeling 
• DAX Measures  
• Data Visualization  

## Dashboard Insights
• Electronics and Beauty contribute significantly to total sales  
• Credit card and UPI are major payment methods  
• Sales show seasonal fluctuations across months
